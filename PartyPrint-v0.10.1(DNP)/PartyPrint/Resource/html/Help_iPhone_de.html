Anschließen von iOS-Geräten (iPhone usw.)

Anschließen Ihres iOS-Geräts

  1. Erfragen Sie erforderlichenfalls bei Ihrem Veranstaltungsplaner den Namen des PartyPrint-Drahtlosnetzwerks und des hierfür benötigten Kennworts.
  2. Bauen Sie eine Verbindung zum PartyPrint-WLAN auf.
  3. Wählen Sie auf Ihrem iOS-Gerät Einstellungen.
  4. Wählen Sie die Option WLAN.
  5. Vergewissern Sie sich, dass die WLAN-Verbindung auf AN steht.
  6. Suchen Sie in der daraufhin angezeigten Liste nach dem Namen des PartyPrint-Netzwerks und tippen Sie diesen an. Sofern das Netzwerk nicht kennwortgeschützt ist, sind Sie nun mit dem Netz verbunden.
  7. Ist der Zugang zum PartyPrint-Netzwerk durch ein Kennwort geschützt, so geben Sie dieses ein und wählen Sie Connect (Verbinden), um mit diesem eine Verbindung aufzubauen (erfragen Sie das ggf. hierfür benötigte Kennwort beim Veranstaltungsplaner).
  8. Öffnen Sie dazu die PartyPrint-App auf Ihrem iOS-Gerät.
  9. Sobald Sie die App-Oberfläche sehen, sind Sie mit dem Server verbunden.

Bei der Übertragung häufig auftretende Probleme

Verbindungsaufbau mit dem WLAN:

  1. Wählen Sie auf Ihrem iOS-Gerät Einstellungen.
  2. Wählen Sie die Option WLAN.
  3. Vergewissern Sie sich, dass die WLAN-Verbindung auf AN steht.
  4. Suchen Sie in der daraufhin angezeigten Liste nach dem Namen des PartyPrint-Netzwerks und tippen Sie diesen an. Sofern das Netzwerk nicht kennwortgeschützt ist, sind Sie nun mit dem Netz verbunden.
  5. Ist der Zugang zum PartyPrint-Netzwerk durch ein Kennwort geschützt, so geben Sie dieses ein und wählen Sie Connect (Verbinden), um mit diesem eine Verbindung aufzubauen (erfragen Sie das ggf. hierfür benötigte Kennwort beim Veranstaltungsplaner).
  6. Öffnen Sie dazu die PartyPrint-App auf Ihrem iOS-Gerät.
  7. Sobald Sie die App-Oberfläche sehen, sind Sie mit dem Server verbunden.

Beim Verbindungsaufbau auftretende Fehlermeldung:

Eine Meldung des Inhalts „The application cannot connect to the server. Please connect to the Wi-Fi network for this event“ (Die Anwendung kann keine Verbindung zum Server aufbauen. Bitte wählen Sie das zur Veranstaltung eingerichtete WLAN) bedeutet, dass die App mit dem Drahtlosnetzwerk und/oder dem Server keine Daten austauschen kann. Um Bilder auf den Server laden zu können, benötigt die App eine funktionierende Verbindung mit dem PartyPrint-Drahtlosnetzwerk. Tippen Sie auf Retry (Erneut versuchen) im Hauptbildschirm von PartyPrint, um einen erneuten Versuch eines Verbindungsaufbaus zu starten.

Neue Bilder aufnehmen

Sobald Sie eine Verbindung mit dem PartyPrint-Netzwerk hergestellt haben, können Sie neu aufgenommene Bilder mit der App übertragen.

  1. Öffnen Sie dazu die PartyPrint-App auf Ihrem iOS-Gerät.
  2. Tippen Sie im Hauptbildschirm auf Take Photo (Foto aufnehmen). Nehmen Sie nun mit der Kamera Ihres iOS-Geräts ein Bild auf. Bearbeiten Sie ggf. die aufgenommenen Bilder in der gewünschten Weise über die Menüs und Einstellungen Ihrer Kamera. (Durch Antippen von Cancel (Abbrechen) gelangen Sie zum Hauptbildschirm von PartyPrint zurück.)
  3. Nachdem Sie das Bild aufgenommen haben, tippen Sie auf Use (Verwenden) um fortzufahren. Durch Antippen von Retake (Aufnahme wiederholen) gelangen Sie zur Kamera-Software zurück und können eine andere Aufnahme anfertigen.
  4. In der Übersichtsdarstellung haben Sie die Möglichkeit, versehentlich gewählte Bilder durch Antippen des X neben dem Bild zu entfernen.
  5. Tippen Sie auf Print (Drucken), wenn Sie Ihre Bilder nun ausdrucken möchten, oder auf Back (Zurück), um zum Hauptbildschirm von PartyPrint zurückzukehren.

Vorhandene Bilder auswählen

Je nach den von ihrem Veranstaltungsplaner gewählten Einstellungen haben Sie darüber hinaus die Möglichkeit, auch bereits vorhandene Bilder zum Ausdrucken auf den PartyPrint-Server zu übertragen.

  1. Öffnen Sie dazu die PartyPrint-App auf Ihrem iOS-Gerät.
  2. Tippen Sie im Hauptbildschirm auf Choose Existing (Vorhandene auswählen).
  3. Wählen Sie aus dem sich daraufhin öffnenden Kameraalbum-Menü das Album, das verwendet werden soll. Führen Sie einen Bildlauf durch das Album durch und wählen Sie das auszudruckende Bild aus.
    (Je nach den Einstellungen sind Sie möglicherweise in der Lage, mehrere Bilder auf einmal zu drucken. Falls Sie dabei zu viele Bilder auswählen sollten, erhalten Sie einen entsprechenden Hinweis.)
  4. Haben Sie Ihre Auswahl abgeschlossen, so tippen Sie auf Done (Fertig).
  5. In der Übersichtsdarstellung haben Sie die Möglichkeit, versehentlich gewählte Bilder durch Antippen des X neben dem Bild zu entfernen.
  6. Tippen Sie auf Print (Drucken), wenn Sie Ihre Bilder nun ausdrucken möchten, oder auf Back (Zurück), um zum Hauptbildschirm von PartyPrint zurückzukehren.

Bei der Übertragung häufig auftretende Probleme

Langsame Übertragung

Geringe Übertragungsgeschwindigkeiten können dann auftreten, wenn viele Veranstaltungsbesucher gleichzeitig auf das Drahtlosnetzwerk zugreifen. Die Übertragung der von Ihnen ausgewählten Bilder beginnt, sobald es das System zulässt. Verzögerungen können auch dann auftreten, wenn mehrere Bilder gleichzeitig übertragen werden.

Netzwerk-/Serververbindungsfehler:

Informationen über Meldung zu Netzwerk-/Serververbindungsfehlern finden Sie unter „Verbindungsfehlermeldungen“. Eine Anleitung dazu, wie Sie eine Verbindung mit dem PartyPrint-WLAN aufbauen, finden Sie unter „Aufbauen einer Verbindung zum WLAN“.

Fehler beim Drucken:

Eine bei einem laufenden Druckvorgang ausgegebene Fehlermeldung kann darauf hindeuten, dass das Netzwerk an seine Kapazitätsgrenze angelangt ist. Tippen Sie in einem solchen Falle noch einmal auf Print (Drucken), um den Vorgang zur Übermittlung Ihrer Bilder neu zu starten.

Vorhandene Bilder lassen sich nicht ausdrucken:

Falls sich vor der Veranstaltung aufgenommene Bilder nicht ausdrucken lassen, so hat der Veranstaltungsplaner diese Funktionalität für die vorliegende Veranstaltung deaktiviert.

Nachdrucken eines Bilds

Sie möchten einen weiteren Abzug von Ihrem Bild ausdrucken? Das letzte Bild war so schön, dass Sie ein Exemplar davon Ihrer Mutter zukommen lassen möchten? Um einen weiteren Abzug anzufertigen, tippen Sie im Hauptmenü von PartyPrint auf Choose Existing (Vorhandene auswählen), wählen Sie einfach das gleiche Bild noch einmal und tippen Sie dann auf Print (Drucken). Kurz darauf halten Sie einen weiteren Abzug in Händen! Weitere Informationen finden Sie unter „Vorhandene Bilder verwenden“
