Connexion des appareils

Connexion de votre appareil

  1. Si besoin, demandez au responsable événementiel de vous fournir le nom du réseau sans fil PartyPrint, ainsi que le mot de passe pour y accéder.
  2. Connectez-vous au réseau sans fil PartyPrint.
  3. Sur votre appareil, accédez aux Paramètres.
  4. Sélectionnez l'option Wi-Fi.
  5. Vérifiez que la connexion Wi-Fi est activée (ON).
  6. Localisez le nom du réseau PartyPrint et sélectionnez-le. Si l'accès au réseau n'est pas protégé par mot de passe, vous êtes connecté au réseau.
  7. Si l'accès au réseau est protégé par mot de passe, saisissez le mot de passe de connexion Wi-Fi au réseau PartyPrint et sélectionnez Join (Rejoindre) pour vous connecter. (Adressez-vous au responsable événementiel pour connaître le mot de passe.)
  8. Ouvrez l'application PartyPrint sur votre appareil.
  9. Lorsque l'écran de l'application apparaît, vous êtes connecté au serveur.

Connexion au réseau Wi-Fi :

  1. Sur votre appareil, accédez aux Paramètres.
  2. Sélectionnez l'option Wi-Fi.
  3. Vérifiez que la connexion Wi-Fi est activée (ON).
  4. Localisez le nom du réseau PartyPrint et sélectionnez-le. Si l'accès au réseau n'est pas protégé par mot de passe, vous êtes connecté au réseau.
  5. Si l'accès au réseau est protégé par mot de passe, saisissez le mot de passe de connexion Wi-Fi au réseau PartyPrint et sélectionnez Join (Rejoindre) pour vous connecter. (Adressez-vous au responsable événementiel pour connaître le mot de passe.)
  6. Ouvrez l'application PartyPrint sur votre appareil.
  7. Lorsque l'écran de l'application apparaît, vous êtes connecté au serveur.

Message d'erreur de connexion :

Le message "The application cannot connect to the server. Please connect to the Wi-Fi network for this event" (La connexion de l'application avec le serveur est impossible. Veuillez vous connecter au réseau Wi-Fi pour cet événement) vous informe d'un échec de communication entre l'application et le réseau sans fil/serveur. L'application doit être connectée au réseau sans fil PartyPrint pour que les images puissent être transférées sur le serveur. Appuyez sur le bouton Retry (Réessayer) de l'écran principal de PartyPrint pour effectuer une nouvelle tentative de connexion.

Transfert d'images

Capture de nouvelles photos

Une fois connecté au réseau PartyPrint, vous pouvez envoyer les nouvelles photos capturées dans l'application PartyPrint.

  1. Ouvrez l'application PartyPrint sur votre appareil.
  2. À partir de l'écran principal, sélectionnez le bouton Take Photo (Prendre une photo). Vous pouvez maintenant prendre une photo avec l'appareil-photo de votre appareil portable. Utilisez les menus et paramètres de votre appareil-photo pour optimiser la qualité de vos photos. (Appuyez sur le bouton Cancel (Annuler) pour revenir à l'écran principal de PartyPrint.)
  3. Une fois la photo prise, appuyez sur le bouton Use (Utiliser) pour poursuivre. Appuyez sur le bouton Retake (Prendre une nouvelle photo) pour retourner au menu de l'appareil-photo et prendre une autre photo.
  4. Sur l'écran de visualisation, vous pouvez supprimer n'importe quelle image sélectionnée par erreur en appuyant sur la croix X à côté de l'image.
  5. Appuyez sur le bouton Print (Imprimer) pour imprimer vos images, ou sur le bouton Back (Précédent) pour revenir à l'écran principal de PartyPrint.

Sélection d'images existantes

Selon le paramétrage effectué par votre responsable événementiel, vous pouvez également transférer sur le serveur PartyPrint des photos existantes pour les imprimer.

  1. Ouvrez l'application PartyPrint sur votre appareil.
  2. À partir de l'écran principal, sélectionnez le bouton Choose Existing (Sélectionner des photos existantes).
  3. Lorsque le menu des albums de votre appareil-photo apparaît, sélectionnez l'album à utiliser. Naviguez dans l'album pour sélectionner l'image que vous souhaitez imprimer.
    (Selon le paramétrage, vous pouvez avoir la possibilité d'imprimer plusieurs images simultanément. Un message apparaît pour vous indiquer, le cas échéant, que vous avez sélectionné un nombre d'images trop important.)
  4. Lorsque vous êtes prêt à continuer, appuyez sur le bouton Done (Terminer).
  5. Sur l'écran de visualisation, vous pouvez supprimer n'importe quelle image sélectionnée par erreur en appuyant sur la croix X à côté de l'image.
  6. Appuyez sur le bouton Print (Imprimer) pour imprimer vos images, ou sur le bouton Back (Précédent) pour revenir à l'écran principal de PartyPrint.

Problèmes fréquents concernant le transfert

Vitesses de transfert trop faibles :

Le transfert peut être lent lorsqu'un nombre important de personnes utilisent le réseau sans fil simultanément. Une fois les images soumises, le transfert s'effectue dès que le système le permet. De même, si vous envoyez un grand nombre d'images simultanément, des délais peuvent être observés.

Erreurs de connexion au réseau/serveur :

Pour plus d'informations sur les messages d'erreur de connexion au réseau et au serveur, consultez la rubrique "Messages d'erreur de connexion." Pour plus d'informations sur la procédure de connexion au réseau Wi-Fi, consultez la rubrique "Connexion au réseau Wi-Fi."

Erreur lors de l'impression :

Une erreur lors de l'impression peut signifier que le réseau sans fil est occupé. Appuyez à nouveau sur le bouton Print (Imprimer) pour relancer vos impressions.

Impression d'images existantes impossible :

Si l'impression de photos prises avant l'événement est impossible, cela signifie que cette fonction a été désactivée par le responsable événementiel pour cet événement.

Réimpression d'une image

Vous souhaitez un deuxième tirage d'une même image ? Vous aimez beaucoup la dernière photo et souhaitez un exemplaire pour votre maman ? Pour effectuer un autre tirage, sélectionnez Choose Existing (Sélectionner des photos existantes) dans le menu principal de PartyPrint, puis sélectionnez la même image et appuyez sur Print (Imprimer). Vous obtenez ainsi en un clin œil un nouveau tirage ! Pour plus d'informations, consultez la rubrique "Utiliser des images existantes."
